# Scan Chan Design Director

Permanent Creative Director skill for Scan Chan.

This skill reviews visual, UX, motion, mascot, brand, and emotional decisions before implementation. It does not replace the project documents; it orchestrates them so every visual implementation can be approved, revised, or rejected against the correct source of truth.

## Contents

- `SKILL.md` - creative review protocol and approval standard.
- `references/design-document-router.md` - which source documents govern each design decision.
- `references/creative-review-checklists.md` - dedicated review checklists for UI, UX, motion, mascot, states, scanner, Home, accessibility, emotion, and brand.
- `agents/openai.yaml` - Codex skill metadata.

## Maintenance

Keep this skill short and procedural. Update source documents when design truth changes. Update this skill only when the review workflow changes.
